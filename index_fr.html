Expérience

Développeur web (Rails, Ember.js)

2012 - 2014 Smartesting, Besançon

En tant que membre de l'équipe R&D Smartesting, j'ai été amené à travailler durant 1 an et demi sur le site Zest testing, une application collaborative facilitant l'écriture de scénarios de tests.
Ma participation s'est portée sur tous les aspects du site: frontend (Ember.js), backend (Ruby on Rails) et administration (Elastic search, Puppet).

Je suis le principal développeur de zest-publisher, l'outil permettant d'exporter les scénarios décrits dans Zest en tant que tests exécutables.

J'ai aussi été le principal développeur de certaines fonctionnalités de l'application:

  • exécution de tâches de fond
  • éditeur de scénarios / actionwords
  • éditeur de table de données
  • refactoring de tests (renommage, extraction et inlining d'actionwords)
  • création et mise en place de zest-publisher as a service afin de permettre l'usage de l'outil en ligne
  • recherche avancée basée sur elastic search

J'ai aussi activement participé à la mise à disposition de code sous licence open-source (zest-publisher, Ember easy datatable, Scalability test)

Développeur web (Plone, Django)

2009 - 2012 Zest software, Rotterdam

J’ai été durant plus d’un an responsable et principal développeur du site Prettig Personeel, un logiciel de gestion de personnel en ligne.
Mon rôle consistait à développer de nouvelles fonctionnalités, refactoriser le code et gérer les releases ainsi que le déploiement sur les sites de test et de production.

J’ai aussi été responsable du développement du site Oldtimers Verzeken qui est un site de vente d’assurance pour les véhicules anciens. A cette occasion, j’ai créé ZestMicroCMS, un système de gestion de contenu basé sur la plateforme Django dédié aux sites pour lesquels Plone est surdimensionné.

J’ai aussi participé à un certain nombre d’autres sites développés sous Plone. Pour Lilly Medical j’ai été chargé de la compatibilité iPad et diverses autres tâches impliquant Javascript.
Pour Minaraad (site permettant de gérer les projets inter-commissions du gouvernement Belge) j’ai créé le système de gestion de projets.
J’ai aussi travaillé sur le projet Vagabonds (site permettant de recenser les infirmières libérales aux États-Unis, le site n’est pas encore en ligne). J’y ai intégré le système permettant aux utilisateurs de remplir leur profil.

Il arrive aussi que des clients contactent Zest software afin de mettre à jour des sites existants. J’ai ainsi été amené à travailler pour VJ Movement (un journal en ligne) afin d’intégrer le nouveau thème, corriger certains bugs et intégrer les comics interactifs dans le site dédié aux illustrations Cartoon Movement.
J’ai aussi procedé à la mise à jour du site Den Haag Ontmoetding (site permettant de gèrer les projets inter-écoles de la Haye) afin de corriger un certain nombre de problèmes.
J’ai créé la troisième version de Franel (une plateforme d’apprentissage du Néerlandais et du Français). Ce site repose sur un service web contenant l’ensemble des données du systèmes (utilisateurs, leçons, résultats) et une base de données MySQL afin de cacher les données et éviter les appels trop fréquents au service web.
J’ai été chargé des mises à jour de ces quatre derniers sites puisque j'étais généralement responsable des projets développés sous Django. J'étais aussi chargé des tâches liées à Javascript.

Doctorat d’informatique (Validation des services web à l’aide de la génération automatique de tests)

2005 - 2009 LIFC, Besançon

J'ai créé la plateforme iTaC-QoS (iTaC Tests and Certifies Quality of Services). Celle-ci permet de valider des services web grâce à la génération de tests à partir de modèles UML.
Cette plateforme permet une automatisation complète du processus de test et de validation, la seule tâche devant être réalisée manuellement est la création du modèle.

Quatre articles concernant iTaC-QoS ont été publiés dans des journaux, conférences et workshop internationaux.
De plus, j’ai effectué de nombreuses heures d’enseignement durant ces trois années, sur des domaines variés (bureautique, EJB, Intelligence Artificielle, Découverte de Java ...)

  • Python
  • Java
  • Web Services
  • Génération de tests
  • UML
  • Test Designer
